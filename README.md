# Asteroid Game

* [The Game](https://paul-surf.github.io/astGame/)

## Controls

```
W = Foward
A = Rotate Left
D = Rotate Right

Space = Shoot
```

## How To Play

The goal of the game, is to shoot all the asteroids, while dodging the them.
